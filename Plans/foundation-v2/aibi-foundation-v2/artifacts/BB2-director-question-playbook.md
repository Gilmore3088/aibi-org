## Patterns I'll watch for in management responses

| Pattern | What it tells me |
|---|---|
| Specific names, dates, numbers | Substance — they have an actual program |
| Specific examples / use cases | Substance — they're tracking what's deployed |
| Repeated buzzwords without specifics | Vague — dig deeper |
| "We're working on it" / "we plan to" | Aspirational — request a date |
| Defensiveness or deflection | Resistant — needs follow-up in committee |
| Real artifacts I can see | Strong — request copies for the file |

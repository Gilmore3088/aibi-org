The Anatomy of System Failure

The six cognitive and structural patterns that lead to AI compromise.

blind

Prompt Blindness

Users stop scrutinizing AI outputs as they become more familiar with the system's fluency.

leak_add

Data Exfiltration

Inadvertent leaking of proprietary data into public training sets via unsecured LLM prompts.

psychology_alt

Recursive Logic Bias

The AI begins reinforcing institutional biases found in historical banking records, creating a feedback loop of discrimination.

model_training

Prompt Injection

Malicious instructions designed to bypass the AI's safety filters or extract hidden data.

warning

Hallucination Drift

When a system confidently generates false financial projections based on incomplete data.

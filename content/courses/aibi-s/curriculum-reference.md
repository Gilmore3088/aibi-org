# AiBI-S Specialist Curriculum Reference

**Source:** AIDB Operators Skills Master Class structure (adapted for community banking)
**Status:** Reference for future AiBI-S course development
**Saved:** 2026-04-16

## Structure: 6-Week Cohort Format

### Weeks 1-2: Foundation

| Learn | Build | Strategize |
|-------|-------|------------|
| Security landscape: risks and known solutions | Enterprise Security Decision Guide | AI Strategy Assistant with context files and first skill |
| Tool and environment choices and enablement | Agent architecture — identity, rules, memory, skills | Context Curation Plan |
| Work decomposition — what makes a task "agent-shaped" | Context curation fundamentals | Work Decomposition Blueprint |

### Weeks 3-4: First Build

| Learn | Build | Strategize |
|-------|-------|------------|
| MCP architecture and enterprise integrations | First agent end-to-end, wired to external services | Enablement Checklist — tools, permissions, MCP inventory |
| Skill design patterns and context engineering | 2+ reusable skills (email triage, briefings, etc.) | Agent Evaluation Playbook |
| Evaluation methodology — what "good enough" means | Agent orchestration and monitoring layer | Governance Framework |

### Weeks 5-6: Scale and Orchestrate

| Learn | Build | Strategize |
|-------|-------|------------|
| Systematic work audit methodology | Second agent — different job, different tools | Organizational Work Audit Guide |
| Use case prioritization and cost-value estimation | Multi-agent system design and connections | Agent-Ready Organization Blueprint |
| Multi-agent orchestration patterns | Personal agent playbook | Digital Workforce Management Playbook |
| Operating model design — roles, lifecycle, scaling | Agent orchestration and monitoring layer | |

## Adaptation Notes for Community Banking

- "Security landscape" maps to AiBI Pillar B: SR 11-7, TPRM, data classification tiers
- "Enterprise integrations" = core banking, document management, M365, compliance tools
- "Agent architecture" needs banking-specific guardrails: never-paste list, ECOA constraints
- "Work audit" methodology should reference the Cornerstone Advisors AI Playbook use cases by department
- "Governance Framework" must align with AIEOG AI Lexicon vocabulary
- 5 role tracks per existing AiBI-S definition: Operations, Lending, Compliance, Finance, Retail
- Live cohort format (not self-paced like AiBI Foundations) — per content/certifications/v1.ts
- Price point: $1,495 per the existing certification spec

## Key Differentiators from AiBI Foundations

- AiBI Foundations: Personal margin of error (learn for yourself, fail privately)
- AiBI-S: Team margin of error (deploy for your department, colleagues catch mistakes)
- AiBI Foundations: Skills are personal productivity tools
- AiBI-S: Skills are institutional assets with governance, versioning, and handoff
- AiBI Foundations: Self-paced, 4-6 hours
- AiBI-S: Live cohort, 6 weeks, process improvement with measured time savings
